1. git clone: Copies an existing Git repository from a remote source to your local machine.
  2. git remote: Manages connections to remote repositories. Commonly used subcommands include:
    1. git remote add: Adds a new remote repository.
    2. git remote remove: Removes a remote repository.
    3. git remote rename: Renames a remote repository.
  3. git fetch: Retrieves changes from a remote repository without merging them into the local branches.
  4. git pull: Fetches changes from a remote repository and merges them into the current branch.
  5. git push: Pushes committed changes from your local repository to a remote repository.
  6. git remote -v: Lists the URLs of the remote repositories associated with your local repository.
  7. git branch -r: Lists the remote branches.
  8. git remote show [remote]: Provides detailed information about a specific remote repository, such as URL, branches, and tracking information.
  9. git push --tags: Pushes annotated tags to the remote repository.
  10. git push [remote] --delete [branch]: Deletes a branch on the remote repository.
  11. git push [remote] :[branch]: Deletes a remote branch.
  12. git push [remote] [local_branch]:[remote_branch]: Pushes a local branch to a remote repository and sets it to track the remote branch.
